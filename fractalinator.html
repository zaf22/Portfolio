Fractalinator

In collaboration with 3 other students, this web application creates parametrized fractal model visuals based on a users input. Users will be able to log into the site, where they can generate fractals and store them in their own fractal library. This information can be accessed any time. Additionally, fractals can be exported into the users choice of image formats.

To deploy the application, we utilized Google Cloud Platform (GCP). We used Flask for backend support, and utilized Cloud Datastore to store the fractals and user information.

Due to the expiration of our subscription to GCP, provided by the University of Pittsburgh, this application is no longer deployed. However, our Github repository is available here, where we provide source code and the responsibilities of each team member.
